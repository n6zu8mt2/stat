ロト6シミュレーター

ロト6のルール説明

ロト6は、1から43までの数字の中から異なる6つの数字を選んで購入する数字選択式宝くじです。1口200円で、抽選では「本数字」6個と「ボーナス数字」1個が選ばれます。以下に、ルールと当選条件を詳しく説明します。

購入方法

1から43までの数字から、異なる6つの数字を選びます。1口200円で、好きな口数を購入できます。

抽選方法

抽選では、1から43までの数字からランダムに6つの「本数字」が選ばれます。さらに、本数字とは異なる1つの「ボーナス数字」が選ばれます。合計で7つの数字（本数字6個＋ボーナス数字1個）が抽選結果となります。

当選条件と賞金

当選は、選んだ6つの数字と抽選結果の一致数に応じて決まります。以下に各等級の条件と賞金をまとめます。

  • 1等（2億円）: 選んだ6つの数字がすべて本数字と一致した場合。
    （例: 選んだ数字「1, 2, 3, 4, 5, 6」が本数字「1, 2, 3, 4, 5, 6」と完全一致）
  • 2等（1000万円）: 選んだ6つの数字のうち5個が本数字と一致し、残りの1個がボーナス数字と一致した場合。
    （例: 選んだ数字「1, 2, 3, 4, 5, 7」、本数字「1, 2, 3, 4, 5, 6」、ボーナス数字「7」）
  • 3等（30万円）: 選んだ6つの数字のうち5個が本数字と一致した場合（ボーナス数字は関係なし）。
    （例: 選んだ数字「1, 2, 3, 4, 5, 7」、本数字「1, 2, 3, 4, 5, 6」、ボーナス数字「8」）
  • 4等（6800円）: 選んだ6つの数字のうち4個が本数字と一致した場合。
    （例: 選んだ数字「1, 2, 3, 4, 7, 8」、本数字「1, 2, 3, 4, 5, 6」）
  • 5等（1000円）: 選んだ6つの数字のうち3個が本数字と一致した場合。
    （例: 選んだ数字「1, 2, 3, 7, 8, 9」、本数字「1, 2, 3, 4, 5, 6」）

注意点

賞金額は理論値であり、実際のロト6では当選者数やキャリーオーバーによって変動します。このシミュレーターでは、賞金を固定値として計算しています。

シミュレーション設定

購入口数を指定して、1回の抽選を行います（1口200円）。
